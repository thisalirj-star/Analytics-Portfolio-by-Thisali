Amazon Sales Dashboard (Power BI)

Overview
This project analyzes Amazon sales data to generate business insights through an interactive Power BI dashboard. The dataset contains transactional sales information primarily related to clothing products sold in India.

Dataset
The dataset was obtained from Kaggle and includes key attributes such as:
- Order details and status
- Product category, size, and SKU
- Sales amount and quantity
- Fulfilment method and shipping details

Due to file size limitations, a sample dataset is included along with a compressed version of the full dataset.

Dashboard Features
The Power BI dashboard consists of five main analytical pages:

1. Sales Overview
   - Revenue, total orders, and sales trends over time

2. Product & SKU Analysis
   - Performance of products based on category, size, and SKU

3. Fulfilment & Logistics Analysis
   - Comparison of fulfilment methods and delivery performance

4. Cancellation Analysis
   - Insights into cancelled orders and revenue impact

5. Geographic Analysis
   - Sales distribution across different regions

Approach
- Cleaned and transformed data using Power Query
- Created calculated measures using DAX (e.g., revenue, cancellation rate)
- Designed interactive visuals with slicers and drillthrough functionality
- Structured the dashboard to support business decision-making

Files Included
- dashboard.pbix → Power BI dashboard file  
- sample_data.xlsx → Sample dataset for preview  
- data.zip → Full dataset (compressed)

Key Outcome
This dashboard enables users to explore sales performance, identify top-performing products, and analyze operational efficiency through interactive and data-driven insights.
