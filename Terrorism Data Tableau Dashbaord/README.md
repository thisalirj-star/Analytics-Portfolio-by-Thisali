Terrorism Data Analysis Dashboard (Tableau)

Overview
This project presents an interactive dashboard built using Tableau to analyze global terrorism data. The dashboard provides insights into incident patterns, casualties, perpetrator behavior, and temporal trends through visually rich and interactive dashboards.

Objective
To analyze terrorism-related data by:
- Identifying patterns in incidents across regions and countries
- Understanding casualty distributions and impact levels
- Analyzing perpetrator groups and target types
- Exploring trends over time

Dataset
- Dataset provided for academic purposes
- Includes:
  - Incident details (date, location)
  - Attack types
  - Casualties (kills, wounded)
  - Perpetrator groups
  - Target types
  - Geographic data

Data Preparation

The dataset was preprocessed before visualization:

Key Steps:
- Removed unnecessary columns
- Cleaned and standardized categorical values
- Converted data types (date, numeric, categorical)
- Handled missing values
- Created calculated fields:
  - Total Casualties
  - Impact Level (Low / Medium / High)
  - Derived date fields for time analysis

Tools Used
- Tableau (Dashboard development & visualization)

Key Insights

- Terrorism incidents are concentrated in specific regions
- Certain attack types contribute to higher casualties
- A few perpetrator groups dominate incident counts
- Trends vary significantly across time and geography

Limitations

- Based on academic dataset
- Missing values required preprocessing assumptions
- No predictive modeling included

Conclusion

This project demonstrates the use of Tableau for building interactive dashboards and extracting insights from complex datasets. It highlights strong visualization and storytelling capabilities in data analysis.
